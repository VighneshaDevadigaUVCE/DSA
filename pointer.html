Pointers are used to hold the adress of a variable. It is used to allocate memory dynamically. Dynamic Memory allocation functions are
  1. malloc
  2. calloc
  3. realloc
  4. free

malloc()[Dynamic Arrays]

  1. Taking array size, n.
  2. Allocating memory in the heap for array, with size n, ptr=new int[5].
  3. Checking availability of memory space, ptr=NULL, then space is insufficient, else read the elements to the array.
